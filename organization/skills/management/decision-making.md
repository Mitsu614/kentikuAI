# スキル: 意思決定

対象: CEO（Jobs）、将来のマネジメント層

## 概要
スタートアップの経営判断を迅速かつ正確に行うためのフレームワーク。

## 原則
- 完璧を待たない。70%の情報で判断する
- 可逆な判断は即決。不可逆な判断は慎重に
- 判断の根拠を必ず記録する（operations/decisions/）

## 意思決定フレームワーク

### 1. 判断の種類を見極める
- **Type 1（不可逆）**: 大型投資、事業撤退、人員削減 → オーナーと相談
- **Type 2（可逆）**: 価格変更、マーケ施策、ツール選定 → CEOが即決

### 2. 判断基準
優先順位：
1. 顧客にとって価値があるか
2. 最初の1件/次の1件に繋がるか
3. 今やる必要があるか（緊急度）
4. 自分たちで実行可能か

### 3. 記録フォーマット
```
# 意思決定 #XXX: タイトル
- 日付:
- 起案者:
- 判断種別: Type 1 or 2
- 決定事項:
- 根拠:
- 期待結果:
- 振り返り予定日:
```

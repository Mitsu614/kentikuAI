Evidence

数字の根拠・前提

営業時に「その根拠は？」と聞かれたら、ここをそのまま説明できます。誇張ではなく、仕組み・出典に基づく前提です。

※1　「数十秒で見積」

見積を生成するAI（Claude）の標準的な応答時間に基づく実測の目安です。工事内容・画像の枚数・通信環境によって変動し、おおむね十数秒〜1分程度です（「○秒」と断定はしていません）。

※2　「搭載データ 約490項目／公的データ3連携」

内蔵の相場データベース：2025-2026年版・750行／約490項目（数値データ行）／68分類（13大分類×55小分類）。構造別・用途別の坪単価、アパマン、解体、設備、外構、仮設、各種工事の単価表を収録。
公的データ（リアルタイム連携）：①e-Stat（政府統計）建設工事費デフレーター・建築着工統計　②国土交通省 不動産情報ライブラリ（地価・取引価格）　③国土交通省 公共工事設計労務単価（職種別日額）。
※「何件・何行」は実装ファイル（cost-reference.ts＝750行）の実数に基づきます。学習で貯まる実績データは各社ごとに増えていきます。

※3　「使うほど精度が上がる／学習」

入力された実績（実際の請求額・原価）から、工種ごとの補正係数を自動で更新する学習機能の“仕組み”によるものです。グラフ（1ヶ月55%→半年94%等）は仕組みを示す概念イメージであり、特定の数値を保証するものではありません。実データが貯まるほど補正が効きます。

※4　料金・利用回数

表示価格は年間契約の金額です。月あたりのAI利用回数の目安はプランごとに設定（デモ：月30回／スタンダード：月50回／プロ：月200回）。デモは無料でお試しいただけます。

※5　「特許審査中」

独自の見積・学習の方式について特許を出願済み・出願審査請求済み＝現在、特許庁にて審査中です。
出願番号：特願2026-100479／出願人：有限会社中野工務店／発明の名称：建築業務支援システム、建築業務支援方法及びプログラム／請求項13。
※審査中であり、登録（特許取得）を確約するものではありません。「特許取得済み（登録済み）」とは表現していません。

建築ブースト - AI建築見積・業務自動化システム

保存場所保存されるデータ説明

2. 法令に基づく場合

法令に基づき開示が求められた場合、人の生命・身体・財産の保護のために必要がある場合、その他個人情報保護法第27条各号に該当する場合には、ユーザーの同意を得ることなく個人情報を第三者に提供することがあります。

第6条（匿名加工情報・学習ループについて）

当社は、本サービスの見積精度を継続的に向上させるため、ユーザーの見積実績データを匿名加工情報として活用します（以下「学習ループ」といいます）。

  1. 学習ループに使用されるデータは、材料費・人件費等の統計値のみであり、特定の個人、企業、または案件を識別できる情報は含まれません。
  2. 匿名加工情報の作成にあたっては、個人情報保護法に定める基準に従い、適切な加工を行います。
  3. 匿名加工情報は、全テナント・全サーバーで共有され、すべてのユーザーの見積精度向上に寄与します。
  4. ユーザーは、学習ループへのデータ提供を希望しない場合、当社までご連絡ください。

第7条（Cookieについて）

本サービスはデスクトップアプリケーションとして提供されるため、Cookie（クッキー）は使用しておりません。

第8条（安全管理措置）

当社は、収集した個人情報の漏えい、滅失またはき損の防止その他の安全管理のために、以下の措置を講じます。

  1. 組織的安全管理措置: 個人情報の取扱いに関する責任者を設置し、取扱い規程を整備します。
  2. 人的安全管理措置: 従業者に対し、個人情報の適切な取扱いについて教育・研修を実施します。
  3. 物理的安全管理措置: 個人情報を取り扱う区域の管理、機器および電子媒体等の盗難防止措置を講じます。
  4. 技術的安全管理措置: アクセス制御、通信の暗号化（TLS/SSL）、アクセスログの記録等の技術的対策を講じます。

ユーザーは、個人情報保護法に基づき、当社に対して以下の請求を行うことができます。

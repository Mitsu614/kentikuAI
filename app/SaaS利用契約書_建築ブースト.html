SaaS利用契約書

有限会社中野工務店（以下「甲」という。）と、利用者（以下「乙」という。）とは、甲が提供するSaaS型ソフトウェアサービス「建築ブースト」（以下「本サービス」という。）の利用に関し、以下のとおり契約（以下「本契約」という。）を締結する。

第1条（目的及びサービスの定義）

  1. 本契約は、甲が開発・提供するAI建築見積・業務管理デスクトップアプリケーション「建築ブースト」（以下「本ソフトウェア」という。）の利用許諾及びこれに付随するサービスの提供条件を定めることを目的とする。
  2. 本サービスは、以下の要素により構成される。
    • Electronベースのデスクトップアプリケーション
    • AI見積機能（Anthropic Claude API及びOpenAI画像生成APIを利用）
    • 業務管理機能（工程管理、日報、発注書、予実管理、安全書類、写真台帳等）
    • ローカルデータベース（SQLite）によるデータ保存
    • 匿名統計データのクラウド送信機能（Supabase基盤）
    • リモートライセンス管理機能（有効/無効切替、AIクレジット制御）
  3. 本サービスの詳細な機能及び利用条件は、別紙料金表及び甲が別途定める仕様書による。

第2条（利用料金及び支払条件）

  1. 乙は、別紙料金表に定めるプランに応じた利用料金を甲に支払うものとする。
  2. 利用料金は月額制とし、毎月末日締め翌月末日払いとする。ただし、甲が別途指定する決済方法による場合はこの限りでない。
  3. 利用料金に係る消費税及び地方消費税は乙の負担とする。
  4. 乙が利用料金の支払を遅延した場合、乙は甲に対し、支払期日の翌日から完済の日まで年14.6%の割合による遅延損害金を支払うものとする。
  5. 既に支払われた利用料金は、本契約に別段の定めがある場合を除き、理由の如何を問わず返還しないものとする。
  6. 甲は、第15条の規定に従い、利用料金を改定することができる。

第3条（利用条件）

  1. 1ライセンスにつき、乙が指定する1台の端末においてのみ本ソフトウェアを利用することができる。
  2. 乙は、利用端末を変更する場合、事前に甲に届け出るものとする。甲はリモートライセンス管理機能により、旧端末のライセンスを無効化し、新端末のライセンスを有効化する。
  3. 各プランにおけるAI利用回数は、別紙料金表に定めるとおりとする。利用回数の上限に達した場合、翌月の利用回数リセットまでAI機能の利用は制限される。
  4. Trialプランの利用回数は通算であり、月次でのリセットは行われない。
  5. 乙は、本ソフトウェアの利用にあたり、自己の責任において適切な動作環境（OS、メモリ、ディスク容量、インターネット接続等）を用意するものとする。

第4条（利用制限及び禁止事項）

  1. 乙は、本ソフトウェアに関し、以下の行為を行ってはならない。
    • 本ソフトウェアの全部又は一部の複製、再配布、貸与、譲渡又は第三者への使用許諾
    • 本ソフトウェアのリバースエンジニアリング、逆コンパイル、逆アセンブル又はその他の方法によるソースコードの解析
    • 本ソフトウェアに組み込まれたAPIキー、認証情報その他のセキュリティ情報の抽出、転用又は不正利用
    • 本ソフトウェアのライセンス管理機能の無効化、回避又は改変
    • AI利用回数の不正な操作又は制限の回避
    • 本サービスのサーバー又はネットワークに過度の負荷をかける行為
    • 他の利用者又は第三者の権利を侵害する行為
    • 法令又は公序良俗に反する目的での利用
    • その他、甲が合理的に不適切と判断する行為
  2. 乙が前項に違反した場合、甲は何らの催告を要せず、直ちにライセンスを無効化し、又は本契約を解除することができる。この場合、甲は乙に対し、損害賠償を請求することができる。

第5条（知的財産権）

  1. 本ソフトウェア及びこれに関連するすべてのプログラム、ドキュメント、デザイン、データベース構造、アルゴリズム、AIモデルの学習データその他一切の知的財産権は、甲又は甲にライセンスを許諾する第三者に帰属する。
  2. 本契約に基づく利用許諾は、乙に対して本ソフトウェアの知的財産権の移転を意味するものではない。
  3. 乙が本ソフトウェアを利用して作成した見積書、発注書、工程表、日報その他の出力物（以下「成果物」という。）に係る権利は、乙に帰属する。
  4. 前項にかかわらず、甲は成果物に含まれる匿名化された統計データを、第6条に定める範囲で利用することができる。

第6条（データの取扱い）

  1. 本ソフトウェアにおける乙の業務データ（見積データ、顧客情報、工事情報等）は、原則として乙の端末内のローカルデータベース（SQLite）に保存され、甲のサーバーには送信されない。
  2. 本ソフトウェアは、サービス品質の向上を目的として、以下のデータをクラウド（Supabase基盤）に送信する。乙はこれに同意するものとする。
    • 匿名化された見積統計データ（金額帯、工種分布等。個人情報・顧客情報は含まない）
    • アプリケーションの起動ログ及び機能利用回数等のアクティビティデータ
    • ライセンス認証及びAIクレジット管理に必要な情報
  3. 甲は、前項の匿名統計データをAI見積精度の向上（学習ループ）に利用することができる。ただし、当該データから個別の利用者又は取引先を特定することはできないものとする。
  4. AI機能の利用に際し、乙が入力したデータの一部は、AI処理のためにAnthropic社及びOpenAI社のAPIサーバーに送信される。甲は、送信データの最小化に努めるものとするが、乙は当該送信について同意するものとする。
  5. 甲は、収集したデータについて、善良なる管理者の注意をもって管理し、本契約に定める目的以外に利用しないものとする。
  6. 乙は、本契約終了時に、ローカルに保存されたデータを自己の責任において管理するものとする。甲は、本契約終了後の乙のデータについて一切の責任を負わない。

第7条（免責及び責任制限）

  1. 本ソフトウェア及び本サービスは、現状有姿（AS IS）で提供されるものであり、甲は、明示又は黙示を問わず、商品性、特定目的への適合性、正確性、完全性その他一切の保証を行わない。
  2. AI機能による見積金額、工期算出、材料提案その他の出力結果は、あくまで参考値であり、最終的な判断は乙の責任において行うものとする。甲は、AI出力結果の正確性、妥当性又は完全性について一切保証しない。
  3. 甲は、本サービスの利用又は利用不能に起因して乙又は第三者に生じた損害（逸失利益、データの喪失、業務の中断等を含むがこれに限らない）について、甲の故意又は重過失による場合を除き、一切の責任を負わない。
  4. 前項にかかわらず、甲が乙に対して損害賠償責任を負う場合、その賠償額は、損害発生時から遡って直近12ヶ月間に乙が甲に対し現実に支払った利用料金の総額を上限とする。
  5. 甲は、第三者が提供するAPI（Anthropic Claude API、OpenAI API等）の障害、仕様変更、サービス停止等に起因する損害について責任を負わない。

第8条（契約期間及び解約）

  1. 本契約の有効期間は、契約締結日から1年間とする。
  2. 期間満了の1ヶ月前までに甲又は乙のいずれからも書面による解約の申出がない場合、本契約は同一条件で更に1年間自動更新されるものとし、以後も同様とする。
  3. 乙は、契約期間中であっても、1ヶ月前までに甲に書面（電子メールを含む。）で通知することにより、本契約を解約することができる。
  4. 前項の場合、解約月の翌月以降の利用料金は発生しないものとする。ただし、解約月までの利用料金は全額支払うものとする。
  5. 甲は、乙が本契約のいずれかの条項に違反した場合、相当の期間を定めて催告し、当該期間内に是正されないときは、本契約を解除することができる。ただし、第4条第2項に該当する場合は、催告を要しない。
  6. 本契約が終了した場合、甲はリモートライセンス管理機能により乙のライセンスを無効化する。乙は本ソフトウェアの利用を直ちに中止するものとする。

  2. 甲及び乙は、相手方が前項の表明保証に違反した場合、何らの催告を要せず、直ちに本契約を解除することができる。
  3. 前項に基づき本契約を解除した当事者は、相手方に対し、当該解除により生じた損害の賠償を請求することができる。

第10条（秘密保持）

  1. 甲及び乙は、本契約に関連して知り得た相手方の技術上又は営業上の秘密情報（以下「秘密情報」という。）を、相手方の事前の書面による承諾なく第三者に開示又は漏洩してはならず、本契約の目的以外に利用してはならない。
  2. 前項の規定は、以下のいずれかに該当する情報には適用しない。
    • 開示の時点で既に公知であった情報
    • 開示後に、受領者の責めに帰すべき事由なく公知となった情報
    • 開示を受けた時点で既に受領者が適法に保有していた情報
    • 正当な権限を有する第三者から秘密保持義務を負うことなく適法に取得した情報
    • 秘密情報によることなく独自に開発した情報
  3. 本条の義務は、本契約終了後も3年間存続するものとする。

第11条（サービスの変更・中断・停止）

  1. 甲は、本サービスの内容を、乙への事前通知（電子メール又はアプリケーション内通知）により変更することができる。ただし、重大な変更の場合は、変更の30日前までに通知するものとする。
  2. 甲は、以下のいずれかに該当する場合、乙への事前通知なく、本サービスの全部又は一部を一時的に中断することができる。
    • サーバー、通信回線その他の設備の保守又は工事のためやむを得ない場合
    • 第三者提供のAPIサービスの障害又は停止が生じた場合
    • 天災地変、戦争、テロ、暴動その他の不可抗力が生じた場合
    • その他、甲が合理的に必要と判断した場合
  3. 甲は、前項の中断により乙に生じた損害について責任を負わない。

    • 操作方法・技術的な問い合わせへの対応
    • データ復旧に関する助言（ローカルバックアップからの復旧支援）

第15条（契約の変更）

  1. 甲は、法令の改正、経済情勢の変動その他合理的な理由がある場合、本契約の内容を変更することができる。
  2. 前項の場合、甲は変更内容及び変更の効力発生日を、効力発生日の30日前までに乙に通知するものとする。
  3. 乙が変更の効力発生日までに異議を述べない場合、乙は変更後の契約内容に同意したものとみなす。
  4. 乙が変更に同意しない場合、乙は効力発生日までに本契約を解約することができる。

本契約は、本契約の対象事項に関する甲乙間の完全な合意を構成し、本契約締結以前に甲乙間でなされた当該事項に関する書面又は口頭によるすべての合意、了解及び表明に優先する。

本契約に定めのない事項又は本契約の解釈について疑義が生じた場合、甲及び乙は、信義誠実の原則に従い、誠実に協議の上解決するものとする。

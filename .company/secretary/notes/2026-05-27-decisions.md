# 意思決定ログ - 2026-05-27

## 決定事項

### 建築見積もり・請求書アプリの開発方針
- **背景**: 建築の間取りから過去の建築歴を参照し、価格・材料費・粗利計算・請求書出力ができるアプリが必要
- **判断**: Tauri + React + SQLite で自分用デスクトップアプリとして開発
- **理由**: 自分だけが使うのでローカル完結が最適。Tauriは軽量でクロスプラットフォーム対応
- **開発フェーズ**:
  - Phase 1: 物件登録、材料マスタ、施工履歴（データの箱）
  - Phase 2: 見積もり・原価計算・粗利自動算出
  - Phase 3: 請求書PDF出力・入金管理
- **フォローアップ**: Phase 1 から着手
